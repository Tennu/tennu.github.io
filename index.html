Event and Promise-driven IRC Bot

Tennu is an IRC framework designed for easily creating reusable and highly functional bots.

In a Nutshell

  • Subscribe to Messages and Commands

    Subscribe to the messages and commands you want using our custom high-powered event emitters and subscribers.

  • No parsing IRC

    Message and command objects come with all relevant data already parsed so that you can focus on your bot's logic, not on parsing irc.

  • Modular

    Module system for keeping your events together.

  • Loggable

    Pass in a logger, and Tennu will log messages to it.
