--- layout: document title: Client Methods ---

The client methods are a conglomerate of methods from various plugins, the socket, and the plugin system. There are also a few client specific methods.

Client Specific Methods

config(param: String) -> Json

This method lets you get a configurable value from the config object passed to the constructor. It can be any subset of JSON (string / number / object / array).

connect() -> this, start() -> this

Starts the Client.

disconnect() -> this, end() ->this

Stops the Client.

Action Plugin Delegates

These methods delegate to the action plugin. They are here so that you don't have to rely on the action plugin in almost every plugin.

  • ctcp [deprecated]

  • ctcpResponse

  • part
  • quit
  • say
  • userhost
  • who
  • whois

  • rawf

Self Plugin Delegates

The Self plugin provides a single method, nickname, that provides the Client's current nickname.
